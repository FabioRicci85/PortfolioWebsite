Project Verhuurwebsite WPL2

31-05-2024

Beschrijving:

Je ontwikkelt een website waar je door jou gekozen producten op verhuurt. Je haalt data op uit je eigen database via je eigen ontwikkelde WebAPI.

Persoonlijke uitdagingen:

Het opdelen van de taken was best moeilijk in het begin. Wie ging wat doen en hoe gingen we dit project aangaan? De materie beheersen dat eigenlijk nog niet gekend was, was een grote stap. Waar documentatie hierover vinden en hoe code begrijpen waar je nog nooit over gehoord hebt.

Wat heb ik bijgeleerd:

Heb meer over databases geleerd en wat WebAPI's inhouden. Ook de opbouw voor de WebAPI, hoe starten met Enteties, de connectie tussen database en backend en hoe we al deze informatie doorgeven naar de frontend om zo een dynamische website op te bouwen die al zijn informatie over de producten elders bewaart.

Project MAUI App

24-05-2024

Beschrijving:

Je ontwikkelt de MAUI App voor Android. In de MAUI App kan de data oproepen via de controller in de Web API door middel van een RestService. Je presenteert een MAUI App die uitgevoerd wordt op een Android Simulator en de data van de database kan oproepen door op een knop te klikken.

Persoonlijke uitdagingen:

Na documentatie over het project te verzamelen dacht ik dat dit snel gemaakt zou zijn. In de werkelijkheid was dit anders dan verwacht en liep ik continu tegen fouten aan. Na twee dagen sukkelen is het uiteindelijk gelukt om iets af te leveren dat aan de noden van "de klant" voldeed.

Wat heb ik bijgeleerd:

Dat ik projecten niet moet onderschatten. Ook al lijkt iets simpel, er kan heel wat complexiteit achter schuilen.

Project C#

05-01-2024

Beschrijving:

Maak een idle game, Cookie Clicker, zoals het voorbeeld online. Om de 2 weken krijgen we een nieuwe itteratie met gevraagde aanpassingen.

Persoonlijke uitdagingen:

Mijn persoonlijke uitdagingen bij het maken van de C# applicatie, het spel Cookie Clicker, waren aanvankelijk groot vanwege zelftwijfel en een gebrek aan kennis over C#. Halverwege het project had ik een diepgaand gesprek met een lector, omdat ik het niet meer zag zitten en het project me te complex leek.

Wat heb ik bijgeleerd:

In dit project heb ik geleerd dat, ondanks de moeilijkheidsgraad van bepaalde onderdelen, het belangrijk is om rustig te blijven en mijn opzoekwerk zorgvuldig aan te pakken. Door deze aanpak kon ik het project aangenamer benaderen en heb ik het zelfs sneller voltooid. Bovendien voegde ik extra features toe om het spel aantrekkelijker en grappiger te maken, althans in mijn ogen.
